# T-023: 結果シェア

| 項目 | 値 |
|---|---|
| フェーズ | Phase 4 |
| 優先度 | P1 |
| 依存 | T-019 |

## 目的
結果をSNSで共有できるようにする（バズ導線）。

## スコープ（実装する）
- 結果画面からシェアテキスト共有：レベル・スコア・作成漢字・ハッシュタグ（書式例：`漢字合体ガチャ｜むずかしい｜スコア142｜作った字：林・明・好・樹 ほか3字 #漢字合体ガチャ`）
- Web Share API を基本、非対応はクリップボードコピーにフォールバック

## 受け入れ条件
- [x] シェアテキストが指定書式で生成される
- [x] Web Share API 対応端末で共有シートが出る
- [x] 非対応でクリップボードコピーにフォールバック

## スコープ外
