## 目的
にがて漢字・図鑑を書き取りプリントとして印刷し、家庭学習・宿題に使える実用機能にする。

## スコープ（実装する）
- 印刷用ビュー（`@media print` CSS）：漢字・読み・意味・なぞり枠（薄字）を整列。
- 範囲指定（学年・にがて・図鑑）して `window.print()`。

## 受け入れ条件
- [ ] 指定範囲のプリント用レイアウトが表示され印刷できる
- [ ] 画面表示を崩さず print 専用スタイルで整う

## 主な変更ファイル
- ui/screens/（PrintView）/ print 用 CSS

## 完了の定義 (DoD)
