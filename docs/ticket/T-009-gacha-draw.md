# T-009: ガチャ抽選

| 項目 | 値 |
|---|---|
| フェーズ | Phase 1 |
| 優先度 | P0 |
| 依存 | T-005, T-006 |
| 関連 | PRD F1 / 機能設計 4.3 |

## 目的
レベル別の出現率と weight で部品を抽選する。床の統計的維持の主役。

## スコープ（実装する）
- `src/domain/gacha/GachaService.ts`：`draw(level, pool, rng)`
  - ①`RARITY_RATES[level]` でレアリティ抽選
  - ②当該レアリティ・レベルの部品から `weight` 重み付き抽選
- weight算出方針の文書化（「scope内で成立する2部品有名漢字の数」に比例）。初期weightはT-004の詰み率検証でチューニング

## 受け入れ条件
- [ ] 多数試行でレアリティ分布が `RARITY_RATES` に統計的に一致
- [ ] weightが大きい部品ほど出やすい
- [ ] `rng` は引数注入（`Math.random` 直書きなし）

## スコープ外
- プールデータの最終チューニング（T-004の検証ループで調整）

## テスト
- 固定シードRNGで分布検証、weightの効きの検証

## 完了の定義 (DoD)
- [ ] ユニットテスト緑、domain純粋性をパス
