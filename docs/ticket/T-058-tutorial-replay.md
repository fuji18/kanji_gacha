## 背景
チュートリアル（T-034）は初回のみ表示され、`tutorialDone` を戻す UI が無い。
子ども向けアプリでは「あとからもう一度見たい」「兄弟に見せたい」が高頻度で発生する。

## 目的
いつでもチュートリアルを見直せる導線を用意する。

## スコープ（実装する）
- Home（または About）に「あそびかたをもう一度みる」ボタンを追加。
- 押下でチュートリアルを再表示する。実装は `tutorialDone` の恒久リセットではなく、
  「一時表示フラグ」を優先検討（再表示を閉じたら以後は出ない＝初回体験と分離）。
- 再表示時もフォーカストラップ・Esc（T-054）が機能すること。

## 受け入れ条件
- [ ] Home/About からチュートリアルを再表示できる
- [ ] 再表示を閉じてリロードしても、初回チュートリアルとして再出現しない
- [ ] E2E: 再表示→最後まで進める→閉じる
