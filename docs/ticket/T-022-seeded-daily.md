# T-022: シードデイリー統合

| 項目 | 値 |
|---|---|
| フェーズ | Phase 4 |
| 優先度 | P0 |
| 依存 | T-014, T-016 |
| 関連 | PRD F8 / 機能設計 4.6 |

## 目的
「今日のお題」をサーバー0台で提供する。

## スコープ（実装する）
- Home/デイリー入口→`SessionManager.start(level, 'daily')`（`mulberry32(dailySeed(todayYmdJst(Date.now())))`）
  - ※ T-016 で「今日のお題」入口は実装済み。現状は暫定で `elementary` 固定（`HomeScreen.startDaily`）。
    本チケットで日替わりレベル決定に置換し、ボタンに対象レベルを併記する。
- デイリーのレベルは日替わり固定、対象レベルを画面に明示（PRD F8）
- 今日のお題のベストをローカル保存・更新表示（`dailyBest`）
- 1日1回基本＋ベスト更新の再挑戦可

## 受け入れ条件
- [x] 同一日付で全プレイヤーが同一ガチャ列（再現性）
- [x] 日替わりでレベル/お題が変わる（JST 0:00基準）
- [x] デイリーベストが保存・表示される
- [x] ランキングは未実装（スコープ外）

## スコープ外
- グローバルランキング（後期・要BE）

## テスト
- E2E：同一日付で2回起動しガチャ列一致
