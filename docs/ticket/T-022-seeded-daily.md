# T-022: シードデイリー統合

| 項目 | 値 |
|---|---|
| フェーズ | Phase 4 |
| 優先度 | P0 |
| 依存 | T-014, T-016 |
| 関連 | PRD F8 / 機能設計 4.6 |

## 目的
「今日のお題」をサーバー0台で提供する。

## スコープ（実装する）
- Home/デイリー入口→`SessionManager.start(level, 'daily')`（`mulberry32(dailySeed(todayYmdJst(Date.now())))`）
- デイリーのレベルは日替わり固定、対象レベルを画面に明示
- 今日のお題のベストをローカル保存・更新表示（`dailyBest`）
- 1日1回基本＋ベスト更新の再挑戦可

## 受け入れ条件
- [ ] 同一日付で全プレイヤーが同一ガチャ列（再現性）
- [ ] 日替わりでレベル/お題が変わる（JST 0:00基準）
- [ ] デイリーベストが保存・表示される
- [ ] ランキングは未実装（スコープ外）

## スコープ外
- グローバルランキング（後期・要BE）

## テスト
- E2E：同一日付で2回起動しガチャ列一致

## 完了の定義 (DoD)
- [ ] デイリーが再現性を持って動作
