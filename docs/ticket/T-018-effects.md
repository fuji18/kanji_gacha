## 目的
ガチャ・合体の手応えを最大化する演出を、依存最小（Canvas2D＋CSS）で実装する。

## スコープ（実装する）
- `src/ui/effects/particleField.ts`：Canvas 2D、単一rAFループ、オブジェクトプール（上限管理）
- 合体成功：白フラッシュ・同心円リング2重・16方向パーティクル・読み/意味表示・スコアフロート
- ガチャ：カプセル降下/振動/開封・レアリティ別グロウ（CSS）
- 失敗：赤シェイク・✕（CSS）
- モーション軽減設定への対応

## 受け入れ条件
- [x] 合体成功/失敗/ガチャの演出が再生される
- [x] 連続成功で演出が同時多発しても60fps維持（プール上限で間引き＝ユニットで担保）
- [x] 判定（ロジック）が演出完了を待たない（Canvas は独自rAFループ・操作は同期後fire-and-forget）
- [x] モーション軽減で簡略化される（prefers-reduced-motion）
