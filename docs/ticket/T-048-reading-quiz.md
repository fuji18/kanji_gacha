# T-048: 読みクイズ／意味当て

| 項目 | 値 |

| フェーズ | Phase 7（レベル・学習拡張） |
| 優先度 | P2 |

## 目的
受動的な学習を能動的にするため、作った直後に読み/意味のクイズを出す。

## スコープ（実装する）
- 合体成功後に「この読みは？」3択、または意味→漢字を組ませる逆問題。
- 選択肢は同レベルの他字から生成（`kanjiView` 利用）。正誤を記録（T-035 と連携可）。

## 受け入れ条件
- [ ] 合体後にクイズが出題され、正誤判定される
- [ ] 選択肢が妥当に生成される
- [ ] スキップ可能

## 主な変更ファイル
- domain（選択肢生成 純粋関数）/ ui（クイズUI）/ SessionManager

## 完了の定義 (DoD)
- [ ] 受け入れ条件を満たし、全品質ゲート成功
