| 項目 | 値 |

## 目的
発見した漢字の収集状況を表示する継続フック。

## スコープ（実装する）
- `src/ui/screens/ZukanScreen.svelte`：発見漢字一覧、各漢字の文字・読み・意味、収集率 ○/N（N＝到達可能primary総数）
- 未発見はマスク表示

## 受け入れ条件
- [x] 発見済み漢字が読み・意味付きで一覧表示
- [x] 収集率が ○/N で表示（分母はN、2,136は目安表記）
- [x] 合体で新規発見すると収集率が増える

## スコープ外
- 並べ替え・部首フィルタ（将来P1）

## テスト
- E2E：新規漢字作成→図鑑で収集率増加

## 完了の定義 (DoD)
- [x] 図鑑が永続データを反映して表示
