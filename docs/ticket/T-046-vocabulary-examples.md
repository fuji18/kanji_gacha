# T-046: 熟語・例文

| 項目 | 値 |
|---|---|
| フェーズ | Phase 7（レベル・学習拡張） |
| 優先度 | P3 |
| 依存 | データ追加 |
| 関連 | docs/ideas/レベル設計と学習機能.md §6(17) |

## 目的
作った漢字の熟語・例文（明→明るい/明日）を示し、語彙学習につなげる。

## スコープ（実装する）
- 漢字→代表熟語/例文の**語彙データを追加**（出典・ライセンス確認）。
- 学習カード／図鑑に1〜2例を表示。

## 受け入れ条件
- [ ] 対応字で熟語/例文が表示される
- [ ] データ未整備の字は非表示でフォールバック

## 課題・前提
- 語彙データの出典・ライセンス（JMdict 等の利用可否）の確認が前提。データサイズに注意。

## 完了の定義 (DoD)
- [ ] 受け入れ条件を満たし、全品質ゲート成功
