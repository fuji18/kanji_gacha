## 目的
低学年が自力で遊べるよう、UI見出し・説明・意味などにふりがなを表示する（前提条件級）。

## スコープ（実装する）
- 設定（localStorage 永続）に「ふりがな表示」トグルを追加。
- UI文言・意味表示などに `<ruby>` でふりがなを付与（固定UI文言は手動ルビ、
  漢字の意味/読みは KANJIDIC2 の読みを利用）。
- 既定はオン（小学生モード時）など初期値を検討。

## 受け入れ条件
- [ ] 設定でふりがな表示をオン/オフでき、リロード後も保持される
- [ ] オン時にUI見出し・意味にふりがなが付く
- [ ] レイアウトが崩れない（行高調整）

## 主な変更ファイル
- domain/types.ts（Settings 拡張）/ data/migrations.ts / StorageRepository / 各 screen

- [ ] 受け入れ条件を満たし、全品質ゲート成功
