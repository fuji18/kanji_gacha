# T-010: スコア・コンボ・称号

| 項目 | 値 |
|---|---|
| フェーズ | Phase 1 |
| 優先度 | P0 |
| 依存 | T-005 |
| 関連 | PRD F4,F6 / 機能設計 4.2,4.7 |

## 目的
画数ベースのスコア計算・コンボ倍率・称号ランクを実装する。

## スコープ（実装する）
- `src/domain/score/ScoreService.ts`：
  - `onSuccess(s, awarded)`：`加点 = floor(画数×1) × コンボ倍率`、コンボ前進（`COMBO_STEPS`）
  - `onMiss(s)`：コンボ倍率を1.0にリセット
- `src/domain/rank/resolveRank.ts`：`RANK_TABLE` の固定閾値でランク名（「上位%」表記は不使用）

## 受け入れ条件
- [ ] 画数が大きい漢字ほど高得点
- [ ] 連続成功で倍率が `1.0→1.5→2.0→3.0`（上限3.0）
- [ ] ミスで倍率が1.0に戻る
- [ ] `resolveRank` が閾値境界で正しいランクを返す

## スコープ外
- スコアのUI表示（T-017）
