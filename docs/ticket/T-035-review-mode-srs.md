## 目的
間違えた／時間のかかった漢字を集中して再出題し、定着を高める（学習の核）。

## スコープ（実装する）
- 永続データに「にがて漢字」記録（ミスした組・未完成で終わった対象字・正答までの所要）。
- 復習モード：にがて漢字を優先して山札（出題）を構成する。
- 簡易 SRS：正解で出やすさを下げ、間違いで上げる（軽量な重み付け）。

## 受け入れ条件
- [ ] にがて漢字が永続記録され、復習モードで優先出題される
- [ ] 正解/不正解で出やすさが調整される
- [ ] リロード後も保持される
